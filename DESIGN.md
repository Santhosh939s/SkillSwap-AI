# UI Style
Modern SaaS, Minimal, Professional
Rounded corners (16–20px)
Soft shadows
Subtle glassmorphism only where appropriate
Excellent spacing
High readability
No visual clutter

# Typography
Use a modern, clean sans-serif font (Inter).
Maintain consistent typography hierarchy.
Prioritize readability over decoration.

# Design Inspiration
Create a premium SaaS interface inspired by Linear, GitHub, Vercel, and Stripe.
The application should feel modern, professional, clean, and enterprise-ready.
Avoid flashy effects, excessive gradients, or gaming-style UI.
